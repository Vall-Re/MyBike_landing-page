# MyBike Landing Page

A modern responsive landing page for the MyBike bicycle store, built according to a Figma design. This project demonstrates clean, scalable markup using HTML5, SCSS, and the BEM methodology, with a strong focus on layout accuracy and adaptability.

## Demo Links
- [Museum Landing Page Demo](https://vall-re.github.io/MyBike_landing-page/)
- [GitHub repository](https://github.com/Vall-Re/MyBike_landing-page)

## Design

The layout is based on a Figma design:
- [Figma Design Link](https://www.figma.com/design/NZQAIydtHo5QkINyGLHNcq/BIKE-New-Version?node-id=0-1&p=f&t=HsYBMPCs0M65C2Wv-0)


## Technologies Used
- **HTML5** — semantic markup
- **SCSS** — modular and maintainable styling
- **BEM** — consistent class naming methodology
- **Flexbox & CSS Grid** — layout structure
- **Responsive Design** — desktop, tablet, and mobile support

## Page Structure

- **Header** (navigation: Home, About Us, Compare Bikes, Details, Contacts)
- Section **Compare Bikes**
- Section **Bike Models**
- Section **The details**
- Call to action — **Book a test ride**
- Section **Contact us**
- **Footer**

## Features

- Responsive bike store landing page
- Navigation menu with main sections
- Bike comparison section
- Bike models showcase with prices
- Product details section
- Contact form and contact information
- No JavaScript used

## Notes

- Built with a focus on pixel-perfect layout
- Suitable for frontend developer portfolios
- Clean structure and easy to scale
